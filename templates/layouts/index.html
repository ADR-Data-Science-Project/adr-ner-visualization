O PROJETO

Inteligência Artificial para a Farmacovigilância em Postagens no Twitter relacionadas a Antidepressivos

Resumo: O impacto psicológico da quarentena, causado pela crise sanitária atribuída à nova Sars-Cov-2, fez com que grande parte da população utilizasse medicamentos para tratamento de depressão. Outra consequência do isolamento social é a manifestação individual de opiniões ou julgamentos em redes sociais, tornando-as uma fonte de dados valiosos.
Objetivo: construção de uma solução computacional capaz de identificar automaticamente relatos de efeitos adversos de medicamentos antidepressivos postados na rede social Twitter.
Método: Criação de uma base de dados com aproximadamente 10 mil postagens coletadas da rede social Twitter. Foram utilizadas técnicas de aprendizado de máquina, reconhecimento de entidades nomeadas, processamento de linguagem natural e grafos para identificar medicamentos e seus respectivos efeitos adversos.

Ferramentas: Classificação de entidades nomeadas (Medicamento e Efeito Adverso) no Doccano, treinamento do algoritmo CRF e utilização de bibliotecas para vizualização (Flask + vis.js + networkx). Armazenados em banco de dados NoSQL (MongoDB).
Resultados: o reconhecimento de entidades nomeadas alcançou valores de f-score superiores a 0,58. Uma aplicação web foi construída para mostrar os resultados alcançados pela solução por meio de um grafo.
Conclusão: A rede social mostrou-se uma fonte de dados poderosa para efeitos de farmacovigilância, pois o modelo de classificador apresentado neste artigo demonstra resultados satisfatórios para as técnicas utilizadas tornando-os promissores.
