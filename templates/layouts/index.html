O Projeto

INTELIGÊNCIA ARTIFICIAL PARA A FARMACOVIGILÂNCIA APLICADA EM TWEETS: MEDICAMENTOS ANTIDEPRESSIVOS

(Flask + vis.js + networkx + MongoDB)

O impacto psicológico da quarentena, causado pela crise sanitária atribuída à nova Sars-Cov-2, fez com que grande parte da população utilizasse medicamentos para tratamento de depressão. Outra consequência do isolamento social é a manifestação individual de opiniões ou julgamentos em redes sociais, tornando-as uma fonte de dados valiosos. Objetivo: construção de uma solução computacional capaz de identificar automaticamente relatos de efeitos adversos de medicamentos antidepressivos postados na rede social Twitter. Método: Criação de uma base de dados com aproximadamente 5.000 postagens coletadas da rede social Twitter. Foram utilizadas técnicas de aprendizado de máquina, reconhecimento de entidades nomeadas, processamento de linguagem natural e grafos para identificar medicamentos e seus respectivos efeitos adversos. Resultados: o reconhecimento de entidades nomeadas alcançou valores de f-score superiores a 0,56. Uma aplicação web foi construída para mostrar a navegação dos resultados alcançados por meio de um grafo. Conclusão: A rede social mostrou-se uma fonte de dados poderosa para efeitos de farmacovigilância, pois o modelo de classificador apresentado neste artigo demonstra resultados satisfatórios para as técnicas utilizadas tornando-os promissores.
